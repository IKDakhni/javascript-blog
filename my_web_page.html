Welcome to javascript

javascript is object based scripting language invented specially for web browsers to make websites more dynamic and attractive

Advantages

1.lesser server interactions
2.Immidiate feedback to the users
3.increased interactivity
4.Richer interface
5.Event driven programming
6.GUIFeatures

Disadvantages

1.Client-side JavaScript does not allow the reading and writing of files.This has been kept for security reasons.
2.Javascript cannot be used for networking applications because there is no such support available.
3.JavaScript does not have any multithreading or multiprocessing abilities.

Applications

i) Web Applications
ii) Web Servers
iii)Native Mobile Applications
iv) Hybrid Mobile Applications

Variables

Variables are the containers that are used to hold data in system memory

Data types

Data types allow the programmer to determine the data and the valid operation on data available in the memory.
Different kinds of data types in JavaScript are
1.Numbers
2.Strings
3.Boolean
4.undefined

CONDITIONAL STATEMENTS

Conditional statements control the flow of a program.
If the condition is true, the program will execute a block of statements and if the condition is false ,
flow will go to an alternate block of statements.
The different types of conditional statements are:
1)if statements
2)if else statement

LOOPING STATEMENTS

Looping statements or iterating statements are used to execute the same block of code a specified number of times.
Different types of looping statements are:
1)WHILE Loop
2)FOR Loop
3)DO WHILE Loop

FUNCTIONS

1)A function is a named block of code which is used to wrap a few statements that should be executed one after the other.
2)Functions allows to easily reuse the code. 3)Functions consists of function name,parameters,the value passed and set of commands.
4)SYNTAX function functionName(parameters) { javaScript commands }

OBJECTS

1)Object is the entity and is the fundamental of Javascript.
2)Objects are variables too.But objects can contain many values.
3)A JavaScript object is a collection of named values.
4)Object values are written as name : value pairs

ARRAYS

Arrays are list-like objects whose prototype has methods to perform traversal and mutation operations.
Neither the length of a JavaScript array nor the types of its elements are fixed.
Since an array's length can change at any time, and data can be stored at non-contiguous locations in the array,
JavaScript arrays are not guaranteed to be dense. This depends on how the programmer chooses to use them.
In general, these are convenient characteristics; but if these features are not desirable for your particular use,
you might consider using typed arrays.
